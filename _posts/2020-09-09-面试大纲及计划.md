#### 序列化

JAVA在创建对象后，会有将对象保存到文件中或者通过网络进行传输的场景，所以就用到了序列化，序列化形式有json、xml、二进制文件等，而当需要从这些文件中读取并转化为对象的操作就是反序列化。

对于fastjson来说，主要依赖JAVA的反射机制完成的序列化和反序列化

- 序列化步骤：
  - 获取JAVA对象类信息、类属性信息、属性类型、访问权限、注解配置等
  - 遍历对象的属性及数据
  - JAVA类型转JSON类型
  - 属性名、属性值按键值对拼接字符串
  
- 反序列化步骤：
  - 解析JSON字符串，合法性检测
  - 根据目标JAVA类型，通过反射获取构造方法、属性信息等。
  - 通过无参构造器创建对象
  - 遍历JSON的键值对，将JSON类型转化为JAVA类型
  - 通过反射给对象属性赋值，返回对象

序列化示例：

``` JAVA
import com.alibaba.fastjson.JSON;
import com.alibaba.fastjson.annotation.JSONField;
import java.util.Date;

public class User {
    private Long id;
    @JSONField(name = "user_name") // 重命名
    private String username;
    @JSONField(format = "yyyy-MM-dd") // 日期格式化
    private Date createTime;

    // 无参构造、getter/setter 省略
    public static void main(String[] args) {
        User user = new User();
        user.setId(1L);
        user.setUsername("test");
        user.setCreateTime(new Date());

        // 序列化：Java 对象 → JSON 字符串
        String jsonStr = JSON.toJSONString(user);
        System.out.println(jsonStr); 
        // 输出：{"createTime":"2025-12-23","id":1,"user_name":"test"}
    }
}
```

反序列化示例：

```java
import com.alibaba.fastjson.JSON;

public class UserTest {
    public static void main(String[] args) {
        String jsonStr = "{\"createTime\":\"2025-12-23\",\"id\":1,\"user_name\":\"test\"}";

        // 反序列化：JSON 字符串 → Java 对象
        User user = JSON.parseObject(jsonStr, User.class);
        System.out.println(user.getUsername()); // 输出：test
        System.out.println(user.getId()); // 输出：1
    }
}
```
